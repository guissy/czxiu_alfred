# czxiu_alfred
彩字秀是一个文字表情图制作网站，以此为基础做了一个Alfred插件，用到了python2和apple script

## 使用方法：
如下图所示：
在 Alfred 输入框写上bb开头和文字
```bash
bb 你要的文字
```
出现下拉框，选择一个，按下回车，等待数秒就可以了（取决于网速）
![预览](https://github.com/guissy/czxiu_alfred/raw/master/how.png)
